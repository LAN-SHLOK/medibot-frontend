DISTRIBUTED MICROSERVICE INFRASTRUCTURE

Production-grade architecture bridging local GPU backends to a globally available edge interface.

Ngrok Secure Tunnels

HTTPS tunnels bridge local GPU-accelerated Python backends to the cloud. Zero-config deployment with automatic SSL and real-time traffic monitoring.

Vercel Edge Frontend

Globally distributed edge-hosted interface for zero-latency user interaction. Automatic CDN caching with 99.99% availability SLA.

Vision Engine (OpenCV)

Real-time injury mapping using HSV color segmentation and contour detection. Processes live camera feeds and uploaded DICOM scans frame-by-frame.

Neural Router (MLP)

Custom Multi-Layer Perceptron Neural Network classifies user intent — Greeting, Info, Symptom — in under 5ms using 384-dimensional semantic vectors.

FAISS Vector Database

High-speed similarity search across 3.5TB of indexed medical literature. Returns the top-K most relevant clinical cases in milliseconds for RAG context.

Whisper Voice Engine

Whisper-tiny running on Float16 CUDA precision with Edge-TTS output. Real-time interruption handling and continuous conversational memory across turns.

THE RAG ENGINE — TECHNICAL SPECS

LLM Stack

Llama 3.3-70B for deep clinical reasoning. Llama 3.1-8B for fast greetings & triage. Llama 4 Scout for multimodal vision tasks.

Live Context Retrieval

MediBot cross-references symptoms with millions of real-time case studies via FAISS cosine similarity. Every answer is grounded in peer-reviewed medical data.

Computer Vision Pipeline

BioMed-CLIP model fine-tuned for grayscale medical textures. Processes X-Rays, MRIs, and CT scans with class activation mapping for interpretable heatmaps.

SECURITY & LICENSING
